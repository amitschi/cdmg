# cdmg
A Game Boy DMG emulator written in C with the goal of running it on the [Cheap Yellow Display](https://github.com/witnessmenow/ESP32-Cheap-Yellow-Display) 

## Rules of the Game

- I will not use any AI tools
- I will not look into other Game Boy emulators code

## Resources 

- [The Game Boy Pan Docs](https://gbdev.io/pandocs/)
